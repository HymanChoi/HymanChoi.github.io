npm常用命令

发表于 2019-01-27 | 分类于 Node.js
字数统计: 380 | 阅读时长 ≈ 1

npm是一个node包管理和分发工具，已经成为了非官方的发布node模块（包）的标准。
有了npm，可以很快的找到特定服务要使用的包，进行下载、安装以及管理已经安装的包。

node的安装分为全局模式和本地模式。
一般情况下会以本地模式运行，包会被安装到和你的应用程序代码的本地node_modules目录下。
在全局模式下，node包会被安装到node的安装目录下的node_modules下。

一个npm包是包含了package.json的文件夹，package.json描述了这个文件夹的结构。

查看帮助命令

npm help

安装node模块

npm install moduleName

安装node模块特定版本

npm install moduleName@1.0.0

全局安装命令

npm install -g moduleName

设定全局安装模式

npm set global=true

查看当前使用的安装模式

npm get global

将模块写入dependencies节点（生产环境）

npm install –save

将模块写入devDependencies节点（开发环境）

npm install –save-dev

检查包是否已经过时

npm outdated

更新node模块

npm update moduleName

卸载node模块

npm uninstall moudleName

会引导你创建一个package.json文件，包括名称、版本、作者这些信息等

npm init

查看当前包的安装路径

npm root

查看全局的包的安装路径

npm root -g

查看npm安装的版本

npm -v

查看当前目录下已安装的node包

npm list

以目录的形式来展现当前安装的所有node包

npm list parseable=true
