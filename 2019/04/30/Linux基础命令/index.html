Linux基础命令

发表于 2019-04-30 | 分类于 操作系统
字数统计: 360 | 阅读时长 ≈ 1

Linux是一套免费使用和自由传播的类Unix操作系统，是一个基于POSIX和UNIX的多用户、多任务、支持多线程和多CPU的操作系统。它能运行主要的UNIX工具软件、应用程序和网络协议。它支持32位和64位硬件。Linux继承了Unix以网络为核心的设计思想，是一个性能稳定的多用户网络操作系统。

mkdir a

作用：在当前目录下创建一个空文件夹’a’

ls

作用：查看当前目录下的文件

ll

作用：ls -l 的简写

cd a

作用：进入当前目录下的 a 目录

pwd

作用：查看当前目录的路径

cd ..

作用：返回上层目录

rm -rf a

作用：删除文件夹 a

vi/vim a.js

作用：编辑 a.js 文件，写入新的内容并保存则会创建；
编辑器命令： i 进入插入模式，ESC 返回原来的模式 :w 保存 :q 退出 :wq 保存并退出

cp a.js a1.js

作用：拷贝 a.js 并存入 a1.js

mv a1.js

作用：将 a1.js 移动到新的文件夹下

rm a.js

作用：删除 a.js

cat a.js

作用：查看 a.js

head a.js

作用：查看头部内容

tail a.js

作用：查看尾部内容

head -n 1 a.js

作用：查看第一行

tail -n 2 a.js

作用：查看后两行

grep ‘2’ a.js

作用：搜索 包含 ‘2’
