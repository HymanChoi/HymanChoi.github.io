SumblimeText3常用插件

发表于 2018-07-11 | 分类于 工具
字数统计: 759 | 阅读时长 ≈ 3

Sublime Text 是一个代码编辑器（Sublime Text 3是收费软件，但可以无限期试用），也是 HTML 和散文先进的文本编辑器。
虽然 SublimeText 本身功能已经非常强大，但是更棒的是有数量可观的插件支持它，给它带来更强大的功能，本文将介绍其中20款常用插件。

AutoFileName

自动填充文件名

All Autocomplete

Sublime Text 默认的 Autocomplete 功能只考虑当前的文件，而 AllAutocomplete 插件会搜索所有打开的文件来寻找匹配的提示词。

Autoprefixer

快捷键：在输入 CSS3 属性后（冒号前）按Tab键
CSS3 前缀补充插件，插件使用CanIUse资料库，能精准判断哪些属性需要什么前缀

Alignment

快捷键：Ctrl + Alt + A
代码对齐

Color Highlighter

它以其真实颜色显示所选十六进制颜色代码（如“#FFFFFF”，“rgb（255,255,255）”，“白色”等）。
此外，插件添加颜色选择器，以轻松修改颜色。

CSS Comments

CSS 快速注释

ConvertToUTF8

用于编辑和保存以 GBK，BIG5，EUC-KR，EUC-JP，Shift_JIS 等编码的文件。

DocBlockr

DocBlocker 是在Sublime平台上开发一款自动补全注释插件，支持 JavaScript (including ES6), PHP, ActionScript, Haxe, CoffeeScript, TypeScript, Java, Apex, Groovy, Objective C, C, C++ and Rust.等众多语言

Emmet

它使用仿 CSS 选择器的语法来生成代码，大大提高了 HTML/CSS 代码编写的速度

Git

一些 git 集成到 sublime 文本的插件

HTML/CSS/JS Prettify

快捷键：Ctrl + Shift + H
通过 node.js 为 Sublime Text 3提供 HTML，CSS，JavaScript 和 JSON 代码格式化程序

JavaScript Completions

js 最基本的 api 快查片段

JSFormat

快捷键：Ctrl + Alt + F
JS 代码格式化插件

Keymaps

可以按功能搜索键盘图，并在可搜索的颜色编码列表中显示所有启用的键盘图

MarkdownEditing

MarkdownEditing 是 Markdown 写作者必备的插件，它可以不仅可以高亮显示 Markdown 语法还支持很多编程语言的语法高亮显示

OmniMarkupPreviewer

快捷键：Ctrl + Alt + O
OmniMarkupPreviewer 用来预览 markdown 编辑的效果，同样支持渲染代码高亮的样式

Package Control

快捷键：Ctrl + Shift + P
功能齐全的包管理器

SideBarEnhancements

Sublime Text 侧栏的增强功能

SublimeCodeIntel
